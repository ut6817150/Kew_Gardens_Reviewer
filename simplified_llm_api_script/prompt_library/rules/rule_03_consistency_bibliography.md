# Bibliography Consistency [Red List Standards Consistency (Section 5.1)]
- Check that all references cited in the text appear in the bibliography
- Check that the sources included in the Bibliography were all cited in the text

Note that some in-text citations may reference personal communication (Pers. comm.) - non-recoverable information from sources like emails, interviews or phone calls. Pers comms do not have to be inlcuded in the bibliography. If you are unsure if an in-text citation, not in the bibliography, may be pers comms, mark severity as low, and recommend a confirmation.
