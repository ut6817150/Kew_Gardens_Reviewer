# RedListGuidelines/tables

## Purpose

This folder contains CSV exports of tables extracted from
`RedListGuidelines.pdf`.

The CSV files are generated by `llm_rag/ii_preprocessed_documents/preprocess_pdfs.py`
and correspond to table-parent and table-row records in the document's
`retrieval_blocks.jsonl` file.

## How These Tables Are Used

The extracted tables support row-level retrieval for detailed guideline
examples. They are less central than the compact criteria tables, but they are
useful when a reviewer asks about how a criterion should be interpreted in
practice.

## Notes

Some extracted table text can be awkward because the source PDF layout is not a
clean spreadsheet. Use these CSVs to inspect what was recovered, but rely on
`retrieval_blocks.jsonl` and downstream evaluation notebooks to judge whether
the rows are useful for retrieval.
