# RL_Standards_Consistency/tables

## Purpose

This folder contains CSV exports of tables extracted from
`RL_Standards_Consistency.pdf`.

The tables support row-level retrieval for assessment consistency, field
checking, and mapping-related review prompts.

## Current Table Inventory

| File | Rows | Columns | Notes |
|---|---:|---:|---|
| `table_001.csv` | 8 | 15 | Assessment-role or field-style table with assessor-related columns. |
| `table_002.csv` | 9 | 9 | Field metadata table, including ESRI field-type style information. |
| `table_003.csv` | 6 | 9 | Presence coding table. |
| `table_004.csv` | 5 | 9 | Origin coding table. |
| `table_005.csv` | 5 | 8 | Seasonality coding table. |
| `table_006.csv` | 11 | 9 | Legend/presence style coding table. |

## Notes

The extracted tables overlap conceptually with mapping standards. During
retrieval, these rows can provide consistency-check context while the mapping
standards document provides more detailed spatial-data rules.
